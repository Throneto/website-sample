## ❓ 常见问题

### Q1: 看不到标签云？
**A:** 清除浏览器缓存，硬刷新（Ctrl+F5）

### Q2: 标签都是一样大小？
**A:** 这是正常的，如果标签频率相同，会显示相同大小

### Q3: 点击标签没反应？
**A:** 检查浏览器控制台是否有JavaScript错误

### Q4: 标签显示"暂无标签"？
**A:** 检查文章数据是否包含tags字段

### Q5: 动画不流畅？
**A:** 检查CPU/GPU使用率，关闭不必要的程序
